Описание:
Программа принимает список списков и находит первый валидный подмассив из двух натуральных чисел A < B. Затем генерируется случайное число N ∈ [A, B] и формируются две лестницы из символа "*".
Функции:
check_input(arr) — проверка валидности подмассива  
get_ladder(valid_arr) — генерация числа N и лестниц  
main(arrays) — поиск валидного подмассива и вывод результата  
Функции не вызываются автоматически.
